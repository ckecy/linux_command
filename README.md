### 絕對路徑和相對路徑 ###

- 絕對路徑: 只要是由根目錄`/`寫起，就是絕對路徑的表示法
- 相對路徑: 不是由根目錄`/`寫起，就是相對路徑的表示法

- `.`: 目前自己所在的目錄
- `..`: 目前自己所在的這個目錄得上一個目錄


**注意**:

絕對路徑的寫法雖然比較麻煩，但是絕對不會有問題。 如果使用相對路徑在程式當中，則可能由於你執行的工作環境不同，導致一些問題的發生。
因此，**在進行工作排程與shell script時，強烈建議使用絕對路徑。**
